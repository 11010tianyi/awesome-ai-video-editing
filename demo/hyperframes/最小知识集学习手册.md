# HyperFrames 最小知识集学习手册

> 基于最小知识集学习法，快速掌握 HyperFrames（HTML→MP4）

---

## 一、最小知识集：3 个核心概念

掌握这 3 个概念，你就能用 HyperFrames 生成视频。

### 1. HyperFrames 是什么

> **HyperFrames = HTML 动画 + Puppeteer 截图 + FFmpeg 合成 MP4**

核心流程：
```
HTML/CSS/GSAP 动画  →  Puppeteer 每帧截图  →  FFmpeg 合成  →  输出 MP4
```

**类比理解**：如果 Remotion 是「React 版 After Effects」，那 HyperFrames 就是「HTML 版 Slidev + 自动渲染」。

### 2. Composition（合成）是什么

> **Composition = 一个 HTML 页面，里面包含所有动画元素和时序**

关键属性（必须掌握）：

| 属性 | 说明 | 示例 |
|------|------|------|
| `data-composition-id` | 合成 ID | `<div data-composition-id="hero">` |
| `class="clip"` | 裁剪容器 | `<div class="clip">` |
| `data-start` | 元素出现时间 | `data-start="0.5"` = 0.5秒 |
| `data-duration` | 元素持续时间 | `data-duration="1.0"` = 1秒 |
| `data-width` | 画面宽度 | `data-width="1920"` |
| `data-height` | 画面高度 | `data-height="1080"` |

### 3. Hywen 的 7 步 Skill 使用

> **Hywen 的 7 步技能是 HyperFrames 的核心工作流**

```
Capture → Design → Script → Storyboard → VO + Timing → Build → Validate
```

| 步骤 | 命令 | 作用 |
|------|------|------|
| 1. Capture | `npx skills add heygen-com/hyperframes` | 安装技能 |
| 2. Design | `/hyperframes` | 编写合成 |
| 3. Script | `/hyperframes-cli` | CLI 操作 |
| 4. Storyboard | `/gsap` | GSAP 动画 |
| 5. VO + Timing | - | 配音+时序 |
| 6. Build | `npx hyperframes render` | 渲染 |
| 7. Validate | `npx hyperframes validate` | 验证 |

**关键区别**：Remotion 用 React 组件，HyperFrames 用 HTML + GSAP 动画。

## 二、边界知识：10 个关键细节

| 编号 | 细节 | 说明 |
|------|------|------|
| 1 | `index.html` | 每个合成是一个 HTML 文件 |
| 2 | GSAP 注册 | 必须 `gsap.registerEffect()` |
| 3 | clip 类名 | 必须使用 `class="clip"` |
| 4 | data-* 时序 | 所有动画时序用 `data-*` 属性 |
| 5 | assets/ | 图片、音频放 assets/ 目录 |
| 6 | 渲染命令 | `npx hyperframes render` |
| 7 | 预览命令 | `npx hyperframes preview` |
| 8 | 输出格式 | MP4, 1920x1080, 30fps |
| 9 | GSAP 动画 | 支持 timeline, stagger, easing |
| 10 | 多合成 | 一个项目可以有多个 Composition |

## 七、Hywen 的 7 步 Skill 详解

### 1. Capture（截图/资源）
```bash
npx skills add heygen-com/hyperframes
```
安装技能后，AI 会自动识别项目中的资产。

### 2. Design（设计）
在 Claude Code/Cursor 中输入：
```
/hyperframes
```
AI 会帮你编写正确的 Composition 代码。

### 3. Script（脚本）
在 Claude Code/Cursor 中输入：
```
/hyperframes-cli
```
AI 会帮你执行 CLI 命令。

### 4. Storyboard（故事板）
在 Claude Code/Cursor 中输入：
```
/gsap
```
AI 会帮你编写 GSAP 动画代码。

### 5. VO + Timing（配音+时序）
添加音频文件到 assets/ 目录，AI 会自动同步时序。

### 6. Build（构建）
```bash
npx hyperframes render
```
渲染出 MP4 视频。

### 7. Validate（验证）
```bash
npx hyperframes validate
```
验证 Composition 是否正确。
